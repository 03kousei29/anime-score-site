# Anime Score Lab

アニメ作品を以下の10項目で100点満点評価し、レーダーチャートで比較できるローカルサイトです。

- ストーリー
- キャラクター
- 世界観
- 演出
- 作画
- 音楽
- 感情
- テーマ
- テンポ
- 独創性

## 起動方法

1. ZIPを展開します。
2. `index.html` をダブルクリックします。
3. ブラウザ上でそのまま利用できます。

## 主な機能

- 作品の新規登録、編集、削除
- 10項目を0〜100点で評価
- 最大6作品のレーダーチャート比較
- 平均点と強みの自動表示
- ブラウザのlocalStorageへの保存
- JSON形式でのバックアップ・復元
- 作品名検索

## 注意

データはブラウザ内に保存されます。
ブラウザのサイトデータを削除すると消える可能性があるため、必要に応じてJSONを書き出してください。
